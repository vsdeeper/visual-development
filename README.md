# visual-development

Configuration platform for visual development. You can preview it [here](https://vsdeeper.github.io/visual-development/).

## Develop

Install dependencies

```
pnpm install
```

Develop components

```
pnpm dev
```

## Build lib mode

```sh
pnpm build:lib
```

It will emit `dts` for components, then build output `.js` and `.cjs` file.

## Publish

```sh
pnpm release
```
